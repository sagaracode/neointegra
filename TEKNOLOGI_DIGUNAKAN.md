# Teknologi yang Digunakan dalam Aplikasi Ini

Aplikasi ini dibangun menggunakan beberapa teknologi utama pada sisi backend dan frontend. Berikut adalah ringkasan teknologi yang digunakan:

## Backend
- **Python**: Bahasa pemrograman utama untuk backend.
- **Script Otomasi & Diagnostik**: Banyak script Python untuk pengecekan, diagnosa, dan perbaikan data.
- **SQL**: Digunakan untuk query dan modifikasi database.
- **Shell Script (sh, ps1)**: Untuk backup, restore, dan automasi deployment.
- **Docker**: Terdapat Dockerfile untuk containerization backend.
- **Virtual Environment (venv)**: Pengelolaan dependensi Python.
- **requirements.txt**: Daftar dependensi Python.

## Frontend
- **React.js 18**: Library JavaScript untuk membangun user interface.
- **JavaScript/JSX**: Bahasa utama untuk frontend.
- **Vite**: Build tool dan development server yang cepat.
- **React Router DOM v6**: Untuk routing dan navigasi.
- **Tailwind CSS 3**: Utility-first CSS framework untuk styling.
- **Framer Motion**: Library untuk animasi smooth dan interaktif.
- **Recharts**: Library untuk grafik dan visualisasi data interaktif.
- **Zustand**: State management yang ringan.
- **Axios**: HTTP client untuk API calls.
- **React Hot Toast**: Toast notifications.
- **@heroicons/react**: Icon library dari Heroicons.
- **@headlessui/react**: Unstyled UI components.

## Ringkasan
Aplikasi ini merupakan **Full-Stack Web Application** dengan:
- **Backend**: Python-based dengan banyak automation scripts
- **Frontend**: Modern React.js dengan Vite, Tailwind CSS, dan Recharts
- **Features**: Dashboard layanan, payment integration, SEO analytics
- **Deployment**: Docker-ready dengan Coolify support
- **Documentation**: Comprehensive markdown files untuk semua aspek

**Tech Stack**: PERN-like stack (Python + React + Node tooling) dengan fokus pada performance, modern UI/UX, dan automation.
